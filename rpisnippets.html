Raspberry PI Snippets

A collection of stuff that I find useful when working with my Raspberry PI.
These are in no way the best or correct way to perform the mentioned action.

Code Snippets

sudo xdg-open / To open the file browser as SU
hostname -I To get the IP address
screen Launch a screen session
screen -x Connect to a screen session
Ctrl+A D Detach from a screen session
screen -ls List current screen sessions
screen -XS [#] quit Kill a detached screen
